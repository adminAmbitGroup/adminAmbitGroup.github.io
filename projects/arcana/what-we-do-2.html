Business Understanding

Understand business objectives

  • Define the opportunity or objectives that describe the project’s purpose
  • Identify expected results
  • Assign team

Understand scope of service

  • Understand scope and resource needs
  • Draft project plan

Assign team

  • Define roles and responsibilities. Assign one leader to be held accountable for the success or failure of the overall service.
  • Set up rules of engagement, such as internal and external communication
  • One of the roles should include a product owner who is ultimately responsible for how well the service meets the needs of its users.

  • Familiarize with data

  • Understand data quality problems

Data Preparation and Modeling

Prepare data

  • Set up developer workspace

Build the service

  • Use code reviews to ensure quality
  • Ensure that the individuals building the service communicate closely
  • Conduct integration testing, if appropriate

Evaluation

Conduct QA testing

  • Set up/prepare test environment
  • Migrate code to test environment
  • Seed test environment with test data
  • Develop a plan for conducting QA testing, to include test cases with expected results whenever possible
  • Establish Requirements Traceability Matrix linking test cases to requirements
  • Conduct QA testing

Conduct User Acceptance Testing

  • Develop a plan for conducting User Acceptance testing, to include test cases with expected results whenever possible
  • Conduct UAT
  • Check usability of service for simplicity and intuitiveness

Deployment

Prepare for production deployment

  • Set up/prepare test environment
  • Migrate code to test environment
  • Seed test environment with test data
  • Develop a plan for conducting QA testing, to include test cases with expected results whenever possible
  • Establish Requirements Traceability Matrix linking test cases to requirements
  • Conduct QA testing

Conduct deployment

  • Conduct quick "shake out", which involves end-to-end testing to ensure that the environment does not crash.
  • Conduct training, if needed.
